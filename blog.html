Question-1: What are the different ways to select an element in the DOM?

There are 4 Ways to Select a Element.
1. Using ID -
const element = document.getElementById("uniqueId");
2. Using Class Name -
const elements = document.getElementsByClassName("className");
3. by Tag -
const elements = document.getElementsByTagName("div");
4. Selecting Using CSS Selectors -
const firstElement = document.querySelector(".className");
const allElements = document.querySelectorAll("div.className");

Question-2: What is the difference between innerHTML, innerText, and textContent ?

​In JavaScript, innerHTML, innerText, and textContent are properties used to access or modify the content of HTML elements, each with distinct behaviors:​
1. [innerHTML]: Handles HTML content. When retrieving, it returns the element's content, including any HTML tags. When setting, it parses the provided string as HTML, rendering elements accordingly.​
2. [innerText]: Focuses on the visible text as rendered on the page. It excludes hidden elements and ignores HTML tags, returning only the text that is visible to the user.​
3. [textContent]: Retrieves or sets all text within an element, regardless of visibility or styling. Unlike innerText, it includes text from hidden elements and does not process HTML tags, treating them as plain text.

Question-3: What is event delegation in the DOM?

​Event delegation is a JavaScript technique where a single event listener is attached to a parent element that will manage event for its child elements and will do all the operation for then.

Question-4: What is event bubbling in the DOM?

​Event bubbling in the DOM is when an event starts at the target element and then propagates up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

ACreate an element: const newDiv = document.createElement('div');
Add content: newDiv.textContent = "Hello!";
Set attributes: newDiv.setAttribute('id', 'myDiv');
Add to DOM: document.body.appendChild(newDiv);
Remove element: newDiv.remove(); or parentElement.removeChild(newDiv);
